## 2026-01-24 - Form Accessibility & Next.js Link
**Learning:** Replacing `<a>` with Next.js `<Link>` for internal navigation requires explicit import and improves client-side navigation performance. Also, wrapping inputs in `<form>` is essential for "Enter to submit" even when using `framer-motion` components.
**Action:** Always check imports when converting tags and ensure interactive form elements are semantic.
